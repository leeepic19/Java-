# 餐饮点餐系统 - 项目汇报

## Slide 1: 封面
- **项目名称**: 餐饮点餐系统 (Food Order System)
- **汇报人**: [您的姓名]
- **日期**: 2025年12月29日

## Slide 2: 项目背景与简介
- **项目背景**: 随着数字化转型，传统餐饮业急需线上点餐解决方案以提升效率。
- **项目目标**: 构建一个轻量级、易扩展、界面友好的餐饮点餐平台。
- **核心用户**:
    - **顾客**: 在线浏览、点餐、查看订单。
    - **商家**: 菜品管理、订单处理、营收统计。
- **设计理念**: 前后端分离开发模式，注重用户体验与系统安全性。

## Slide 3: 系统架构设计
- **架构模式**: B/S (Browser/Server) 架构
- **设计模式**: MVC (Model-View-Controller) 分层设计
    - **表现层 (Controller)**: 处理 HTTP 请求，参数校验。
    - **业务层 (Service)**: 核心业务逻辑（订单流转、库存控制）。
    - **持久层 (DAO)**: 数据持久化，与 MySQL 交互。
- **交互方式**: AJAX 异步通信，JSON 数据格式。

## Slide 4: 后端技术栈
- **核心框架**: Spring Boot 3.4.0 (快速开发，自动配置)
- **开发语言**: Java 17 (LTS版本，高性能)
- **ORM框架**: MyBatis-Plus 3.5.5 (简化 SQL 操作，支持 Lambda)
- **数据库**: MySQL 8.0 (稳定可靠的关系型数据库)
- **工具库**:
    - Lombok (简化样板代码)
    - MySQL Connector/J 8.2.0 (修复高危漏洞)
- **安全机制**: 自定义过滤器防御路径遍历攻击 (CVE-2025-41242)。

## Slide 5: 前端技术栈与UI优化
- **基础技术**: HTML5, CSS3, JavaScript
- **核心库**: jQuery 3.6.4 (DOM 操作，AJAX 封装)
- **UI 设计**:
    - **风格**: 现代简约风格，卡片式布局。
    - **字体**: Google Fonts (Noto Sans SC) 优化中文显示。
    - **图标**: FontAwesome 6.4.0 矢量图标库。
- **交互亮点**:
    - 商家轮播展示 (Carousel)。
    - 玻璃拟态 (Glassmorphism) 登录界面。
    - 响应式设计 (适配移动端与桌面端)。

## Slide 6: 数据库设计
- **User (用户表)**:
    - 存储用户基本信息、角色（商家/顾客）。
- **Food (餐品表)**:
    - 关联商家，存储菜品名称、价格、库存、图片。
- **OrderInfo (订单表)**:
    - 核心交易数据，记录下单人、餐品、数量、状态、时间。
- **UserAddress (地址表)**:
    - 用户收货地址管理。
- **关系**: 1个商家 -> N个餐品；1个用户 -> N个订单。

## Slide 7: 核心功能 - 顾客端
- **账户管理**: 注册、登录、退出。
- **浏览体验**:
    - 按商家分组轮播展示。
    - 查看餐品详情（图片、价格、实时库存）。
- **交易流程**:
    - 购物车/直接下单。
    - 填写收货信息（姓名、电话、地址）。
- **订单追踪**:
    - 实时查看订单状态（待处理/已发货/已完成）。
    - 确认收货。

## Slide 8: 核心功能 - 商家端
- **工作台**: 商家专属仪表盘。
- **餐品管理**:
    - 图片上传、信息编辑。
    - 上架/下架控制、库存调整。
- **订单处理**:
    - 实时接收新订单通知。
    - **发货**: 填写预计送达时间。
    - **拒单**: 异常情况处理（自动回滚库存）。

## Slide 9: 项目代码结构
- `src/main/java`: 后端源码
    - `config`: 安全与Web配置
    - `controller`: API 接口定义
    - `service`: 业务逻辑实现
    - `entity`: 数据库实体类
    - `mapper`: 数据库接口
- `src/main/resources`: 资源文件
    - `static`: 前端页面 (HTML/CSS/JS)
    - `application.yml`: 系统配置

## Slide 10: 安全性与稳定性
- **漏洞修复**:
    - 升级 MySQL 驱动至 8.2.0。
    - 升级 Spring Boot 至 3.4.0。
- **安全防护**:
    - 实现 `PathTraversalProtectionFilter` 拦截非法路径请求。
- **异常处理**:
    - 全局异常捕获，统一返回 JSON 格式错误信息。
    - 前端网络异常自动提示。

## Slide 11: 总结与展望
- **项目成果**: 完成了功能闭环的餐饮点餐系统，界面美观，运行稳定。
- **技术亮点**: 全栈开发，前后端分离思想，注重安全。
- **未来展望**:
    - 引入 Redis 缓存热点数据。
    - 增加 WebSocket 实现消息实时推送。
