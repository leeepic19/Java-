# 网上订餐系统启动说明

## 1. 环境配置详情
本项目在当前环境下的具体配置如下，请确保使用以下路径和版本：

### Java 环境 (JDK 17)
*   **版本**: OpenJDK 17.0.16
*   **安装路径**: `C:\Users\35278\.jdk\jdk-17.0.16`
*   **配置命令**:
    ```powershell
    $env:JAVA_HOME="C:\Users\35278\.jdk\jdk-17.0.16"
    $env:Path="$env:JAVA_HOME\bin;$env:Path"
    ```

### Maven 环境 (Maven 3.9.12)
*   **版本**: Apache Maven 3.9.12
*   **安装路径**: `C:\Users\35278\.maven\maven-3.9.12(1)`
*   **可执行文件**: `C:\Users\35278\.maven\maven-3.9.12(1)\bin\mvn.cmd`

### MySQL 数据库 (MySQL 8.0)
*   **版本**: MySQL 8.0.44
*   **安装路径**: `C:\Program Files\MySQL\MySQL Server 8.0`
*   **服务名称**: `MySQL80`
*   **连接信息**:
    *   Host: `localhost`
    *   Port: `3306`
    *   Database: `food_order_system`
    *   User: `root1`
    *   Password: `208716`

## 2. 数据库准备
本项目依赖 MySQL 数据库，启动前请确保：
1.  MySQL 服务已启动。
2.  数据库 `food_order_system` 已创建并初始化。
3.  配置文件 `src/main/resources/application.yml` 中的数据库账号密码正确。

> **注意**：菜品数据存储在本地 MySQL 数据库中，**重启电脑或重启项目后数据依然存在**。

## 3. 启动项目
在项目根目录 (`food-order-system`) 下打开终端 (PowerShell)，执行以下命令：

```powershell
# 1. 设置 Java 环境变量
$env:JAVA_HOME="C:\Users\35278\.jdk\jdk-17.0.16"
$env:Path="$env:JAVA_HOME\bin;C:\Users\35278\.maven\maven-3.9.12(1)\bin;$env:Path"

# 2. 启动 Spring Boot 项目
mvn spring-boot:run
```

项目启动成功后（控制台看到 `Started FoodOrderSystemApplication`），打开浏览器访问：

*   **首页地址**: [http://localhost:8080](http://localhost:8080)

## 6. 常见问题
*   **图片无法显示**：请确保图片已上传到 `C:\temp\food-order-system\upload` 目录。
*   **端口被占用**：如果启动报错 `Port 8080 was already in use`，请关闭占用该端口的进程或修改 `application.yml` 中的 `server.port`。
